# Farm Merge Valley

Farm Merge Valley adalah game yang saya mainkan di platform Discord saat sedang gabut. Saya salah satu mentor didalam [Official Server Farm Merge Valley](https://discord.gg/farm-merge-valley).

---

## Catatan Kaesa (Jangan Menyesal seperti Kaesa)
Guide EN: https://discord.com/channels/1158408201344135258/1372987259665518632

- Tidak tahu kalau coin, gems, dan item lainnya dapat di merge (gabungkan)
	- Membuat saya kehilangan kesempatan dapat `Bonus Merge 5` & `Lucky Merge`
- Tidak tahu bahwa `merge 5` item itu lebih baik daripada `merge 3`
	- Merge 5 item itu hasilnya 2 item tier atasnya, bisa jadi Anda mendapatkan bonus Lucky Merge Up to 2 item lagi, jadi totalnya bisa 4 item.
	- Lucky Merge +1 itu basisnya dari hasil merge. Misal anda merge 5 = 2 hasil, maka kamu bisa mendapatkan +2 bonus lucky merge, jika anda merge 3 = 1 hasil yaa maka kamu hanya bisa mendapatkan +1 bonus lucky merge. 
	- Merge 5 = 2 (+2 Bonus Lucky Merge) = Up to 4.
	- Merge 3 = 1 (+1 Bonus Lucky Merge) = Up to 2.
- Tidak tahu apa itu "Trade with energy" 
	- Guide EN: https://discord.com/channels/1158408201344135258/1282206400063471626
- Tidak tahu jika kita bisa menyimpan energy dengan toolbox 🧰, atau tree dan stone yang dibeli di marketplace.
	- Guide EN: https://discord.com/channels/1158408201344135258/1390383975045922866
	- Catatan: Tree dan Stone yang dibeli di marketplace dapat dipindahkan (movable) akan tetapi tidak dapat di shovel ataupun di simpan di storage. Jadi pastikan beli dan simpan dalam bentuk bubble daripada pop bubble.
- Tidak tahu bahwa kita bisa visit pemain lain tanpa mengonsumsi tiket.
	- Guide EN: https://discord.com/channels/1158408201344135258/1304965686493446146
	- Saat mengunjungi peternakan pemain lain anda tidak click icon apapun (📦 crate, ⚡ energy, 💗 Animal, 💧 Plant, 🟡 Coin (Decoration/Building Truck), 💪 Heavy Object) anda tidak akan mengonsumsi tiket saat kembali.
- Membuka peti/chest ketika peternakan tidak banyak ruang/slot/space (low level farm).
	- Guide EN: https://discord.com/channels/1158408201344135258/1352060826550272032/1355325707210723338
	- Jika anda membuka peti ketika level anda di bawah lv20 maka anda tidak akan mendapatkan gazebo part. jadi saya sarankan shovel atau hapus chest jika anda masih lv20 ke bawah.
	- Dan jika anda membuka peti/chest di saat anda kekurangan ruang, anda kemungkinan akan kesulitan dalam melakukan merge item lain, soalnya saat memuka peti bisa jadi 12 ruang/slot terpakai untuk item reward-nya
- Tidak shovel chest untuk mendapatkan ruang/slot lebih.
	- Anda tahu gold chest sebaiknya anda shovel jika tidak memiliki kunci, karena itu akan memakan ruang anda. Jadi saya sarankan shovel gold chest jika kamu tidak ada kunci. Dan jika anda memiliki bronze dan silver chest anda bisa merge mereka menjadi gold chest untuk mendapatkan sedikit EXP.
- Tidak join discord server dan memanfaatkan fitur visit untuk menghancurkan heavy object seperti ban, haybale, dan traktor. 
	- Guide EN: https://discord.com/channels/1158408201344135258/1374582668842242088
	- Jangan gunakan gems untuk menghancurkan heavy object. manfaatkan fitur visit pemain lain.
- Menggunakan tools 🛠 untuk memperbaiki dekorasi Outhouse, Chicken Cop, dan Windmill.
	- Guide EN: https://discord.com/channels/1158408201344135258/1303396835070578708
	- Jangan gunakan tools untuk memperbaiki hal tersebut karena buang-buang resource. Sebaiknya anda gunakan tools untuk memperbaiki building truck terlebih dahulu.
