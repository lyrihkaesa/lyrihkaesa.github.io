## Font Weight

Ada dua penamaan dalam font weight yaitu `number` dan `string`.

| `number` | `string` (Name Aliases) |
| -------- | ----------------------- |
| 100      | Thin                    |
| 200      | ExtraLight              |
| 300      | Light                   |
| 400      | Regular                 |
| 500      | Medium                  |
| 600      | SemiBold                |
| 700      | Bold                    |
| 800      | ExtraBold               |
| 900      | Black                   |
